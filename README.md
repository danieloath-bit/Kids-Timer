# Kids Timer

A simple visual countdown timer designed for children's everyday tasks.

Built to help kids understand time through a clear, easy-to-read display. Useful for routines like getting ready, brushing teeth, cleaning up, reading, screen-time limits, or transition warnings.

No ads. No clutter. Just a calm, child-friendly timer.

## Features

- Visual analogue-style countdown clock
- Adjustable minutes and seconds
- Start, pause, reset, and replay controls
- Optional ticking sounds with a mute button
- Celebration animation and finish chime when time runs out
- Phone-optimized layout

## Build

Open the project in Android Studio, or run:

```powershell
.\gradlew.bat assembleDebug
```
